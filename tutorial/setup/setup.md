# Setup Unity and Vuforia

## MacOS

### Install [Unity 3D](https://unity3d.com/)

  `brew cask install unity`

### Launch Unity

  `open /Applications/Unity/Unity.app`

### Sign in (or skip it)

  ![Unity Sign in screenshot](screenshots/1_signin.png)

### Create new project
